# Repository Setup Checklist

Use this checklist when moving the package into a GitHub repository.

## Required Repository Files

- `.github/workflows/discover-languages.yml`
- `main.py`
- `requirements.txt`
- `config.py`
- `models.py`
- `discovery/`
- `language/`
- `reporting/`
- `input_examples/domains.csv`
- `docs/GITHUB_ACTIONS_OPERATOR_GUIDE.md`

## Recommended Repository Settings

- GitHub Actions enabled
- Artifact downloads allowed for intended users
- Private repository if domain lists or output should remain internal
- Branch protection as appropriate for production use

## Smoke Test

Run the workflow manually with:

```text
example.org
```

Expected primary output:

```text
language_reports.csv
```

The row should include `example.org` and a language count generated by the engine.
